Privacy Policy for WorkPilot

Effective Date: Upon installation
Last Updated: Today

WorkPilot (“the Extension”, “we”, “us”) is a productivity assistant built for Zoho Cliq. This Privacy Policy explains what data we collect, how it is used, and how users may control their data.

1. What Personal Data We Collect

A. Mandatory (core features)

  • Zoho User ID
  • Timezone
  • User preferences

B. Optional (only if indexing is enabled)

  • Channel ID where bot is added
  • Message metadata (permalink, timestamp, snippet)
  • Tokenized text (not full text unless Deep Indexing enabled)
  • Attachment metadata (file type, name, link)

C. Audit Logs

  • User ID performing the action
  • Action type
  • Timestamp

2. How We Use Your Data

  • Day planning
  • Task creation
  • Meeting summaries
  • Search & provenance
  • Message scheduling
  • Bot orchestration

3. Data Retention

  • Indexer metadata: 90 days
  • Audit logs: 90 days
  • User settings: until deleted by user
  • Scheduled messages: removed after execution

4. User Control

  • Enable/disable indexing
  • Delete indexed data
  • Revoke permissions
  • Reset preferences
  • Remove WorkPilot from channels

5. Security

WorkPilot uses Zoho’s secure infrastructure, encrypted transmission, strict permission scopes, and Catalyst DataStore security.

6. Compliance

This extension complies with GDPR, DPDP Act, and Zoho Marketplace policies.
